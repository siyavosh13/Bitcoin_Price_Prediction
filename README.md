### پیش‌بینی قیمت بیت‌کوین با استفاده از یادگیری عمیق

این پروژه یک مدل پیش‌بینی قیمت بیت‌کوین است که با استفاده از داده‌های تاریخی **یک سال قیمت بیت‌کوین** آموزش دیده و با مشاهده **قیمت ۶ روز گذشته، قیمت روز هفتم** را پیش‌بینی می‌کند.

مدل بر اساس الگوهای زمانی (Time Series) آموزش داده شده و می‌تواند روند کوتاه‌مدت قیمت را تخمین بزند.

# ویژگی‌های اصلی پروژه

- استفاده از **دیتاست قیمت یک‌سال بیت‌کوین**
- پردازش و آماده‌سازی داده‌ها (Cleaning, Scaling, Sequencing)
- استفاده از **Sequence Modeling** جهت پیش‌بینی سری‌های زمانی
- پیش‌بینی قیمت روز هفتم با مشاهده ۶ روز قبل
- رسم نمودارهای:
  - قیمت واقعی vs پیش‌بینی‌شده
  - روند آموزش مدل
- ذخیره مدل نهایی برای استفاده مجدد
